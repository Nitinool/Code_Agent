# 贪吃蛇游戏 (Snake Game)

一个基于HTML5 Canvas的现代化贪吃蛇游戏，支持键盘和触摸控制。

## 项目结构

```
snake-game/
├── index.html          # 主页面
├── style.css           # 样式文件
├── static/
│   ├── css/            # CSS样式文件
│   ├── js/             # JavaScript文件
│   └── images/         # 图片资源
├── docs/               # 项目文档
└── README.md           # 项目说明
```

## 功能特性

- 🎮 键盘方向键控制
- 📱 移动端触摸按钮控制
- 🎯 得分系统和最高分记录
- ⏸️ 游戏暂停/继续功能
- 🔄 重新开始功能
- 📱 响应式设计，支持移动设备
- 🎨 现代化UI设计

## 游戏说明

1. 使用方向键或点击屏幕按钮控制蛇的移动
2. 吃到红色食物可以增加分数和蛇的长度
3. 撞到墙壁或自己的身体游戏结束
4. 按空格键可以暂停/继续游戏

## 技术栈

- HTML5 Canvas
- CSS3 (Grid, Flexbox, 动画)
- Vanilla JavaScript (ES6+)
- LocalStorage (最高分记录)

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
